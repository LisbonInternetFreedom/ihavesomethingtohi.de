Installer Linux

Pourquoi ? Des logiciels libres pour limiter la surveillance sur les appareils utilisés pour rechercher sur internet.

Vous pouvez installez différentes distributions GNU/Linux. C’est le nom donné au différents types de Système d’exploitation GNU/Linux (liste non-exhaustive) : - Distributions libres destinées au grand public - Ubuntu - Debian - Mageia - Linux Mint - Fedora - Distributions libres conseillées par le FSF (Fondation des logiciels libres) : - Trisquel - Parabola - gNewSense - Dragora - PureOS - Distributions libres à usage spécifique : - Kali - Tests d’intrusion / hacking - SteamOS - Jeu en ligne

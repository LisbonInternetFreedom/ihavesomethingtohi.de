# Get started

```
gem install bundler
bundle install
npm install
npm install -g grunt
npm install -g grunt-cli
grunt serve

#Hack it, fix it!
grunt deploy
```

# Translation

Important files :
 - `app/_i18n/en.yml`
 - `app/_en_*`

If you want to add a language:
 - `cp app/_i18n/en.yml app/_i18n/pt.yml`
 - `cp -R app/_en_* app/_pt_*`
 - add you language in `config.yml`
